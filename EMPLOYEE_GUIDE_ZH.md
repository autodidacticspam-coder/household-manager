## 开始使用

### 安装 TestFlight (iOS)

TestFlight 是苹果官方的应用测试工具，用于在应用发布到 App Store 之前进行测试。您需要它来在 iPhone 或 iPad 上安装 Household Manager。

**第一步：下载 TestFlight**
1. 在您的 iPhone 或 iPad 上打开 **App Store**
2. 点击底部的**搜索**标签
3. 在搜索栏中输入 **"TestFlight"**
4. 寻找带有**蓝色飞机图标**的苹果应用
5. 点击**获取**（如果您之前下载过，则点击云图标）
6. 使用 Face ID、Touch ID 或您的 Apple ID 密码进行验证
7. 等待下载完成

**第二步：打开 TestFlight**
1. 安装完成后，点击**打开**或在主屏幕上找到 TestFlight 应用
2. 首次打开时，您可能会看到欢迎屏幕 - 点击**继续**

### 通过 TestFlight 安装应用

安装 TestFlight 后，您将收到测试 Household Manager 的邀请。

**方法一：邮件邀请**
1. 检查您的邮箱，查找来自 TestFlight/Apple 的邀请
2. 在您的 iPhone 或 iPad 上打开邮件
3. 点击**"在 TestFlight 中查看"**或**"开始测试"**按钮
4. TestFlight 将自动打开
5. 点击**接受**加入测试版
6. 点击 Household Manager 旁边的**安装**
7. 等待应用下载和安装
8. 点击**打开**启动应用

**方法二：邀请链接**
1. 您的经理会发送一个 TestFlight 链接（以 `https://testflight.apple.com/...` 开头）
2. 在您的 iPhone 或 iPad 上点击该链接
3. 如果提示，点击**在 TestFlight 中打开**
4. 点击**接受**加入测试版
5. 点击**安装**下载应用
6. 安装完成后，点击**打开**

**方法三：兑换码**
1. 打开 TestFlight
2. 点击右上角的**兑换**
3. 输入经理提供的代码
4. 点击**兑换**
5. 点击 Household Manager 旁边的**安装**

### 使用网页版

如果您更喜欢在电脑或 Android 设备上使用该应用，可以通过网络浏览器访问：

1. 打开任意网络浏览器（Chrome、Safari、Firefox、Edge）
2. 访问：**https://household-manager-two.vercel.app**
3. 使用您的凭据登录（见下一节）

**提示：** 在手机上，您可以将网站添加到主屏幕以便快速访问：
- **iPhone/Safari：** 点击分享按钮 → "添加到主屏幕"
- **Android/Chrome：** 点击菜单（3个点）→ "添加到主屏幕"

## 登录

### 首次登录

1. 打开 Household Manager 应用或网站
2. 您将看到登录屏幕
3. 输入您经理注册的**电子邮件地址**
4. 点击**继续**
5. 检查您的邮箱，查找登录链接（无密码登录）
6. 点击邮件中的链接完成登录
7. 您将自动登录并重定向到应用

### 返回用户

- 应用会记住您的登录状态7天
- 如果您退出登录，只需重复上述登录过程
- 为了快速访问，请使用相同的设备和浏览器

## 我的任务

这是您花费大部分时间的地方 - 查看和完成分配给您的任务。

### 查看任务

1. 在导航中点击**我的任务**
2. 您将在顶部看到两个标签：
   - **已分配** - 您需要完成的任务
   - **已完成** - 已完成的任务

3. 每个任务卡片显示：
   - 任务标题
   - 类别（颜色编码）：儿童看护、清洁、烹饪、洗衣等
   - 优先级：低、中、高或紧急
   - 截止日期/时间
   - 描述（点击展开）

### 完成任务

1. 在**已分配**标签中找到任务
2. 点击**完成**按钮（勾选图标）
3. 任务移至**已完成**标签

### 查看任务详情

1. 点击任何任务卡片查看完整详情
2. 您可能会看到：
   - 完整描述
   - 视频说明（如果有附件）
   - 开始和结束时间（对于定时任务）
   - 经理的备注

### 撤销已完成的任务

犯了错误？您可以撤销已完成的任务：
1. 转到**已完成**标签
2. 找到该任务
3. 点击**撤销**将其移回已分配

## 我的日历

以日历格式查看您的日程、任务和已批准的休假。

### 浏览日历

1. 在导航中点击**我的日历**
2. 使用顶部的**箭头**在月份之间切换
3. 点击**今天**返回当前日期

### 日历视图

您可以使用顶部的按钮切换不同视图：
- **日** - 专注于单天
- **2天** - 两天视图（适合移动设备）
- **周** - 详细的周视图
- **月** - 整月概览

### 理解日历颜色

- **任务** - 按类别颜色编码
- **休假** - 显示您已批准的假期
- **事件** - 特殊日期和活动

### 点击事件

- 点击任何事件查看详情
- 对于任务，您可以直接从日历中标记为完成

## 请假申请

申请休假、病假或其他假期。

### 提交新申请

1. 在导航中点击**请假**
2. 点击 **+ 新申请** 按钮
3. 填写表单：
   - **类型：** 选择休假或病假
   - **选择休假日期：** 点击日历选择您想休假的日期
     - 点击日期选择（再次点击取消选择）
     - 您可以选择多个不连续的日期
     - 选中的日期在日历上方显示为标签
     - 使用"清除全部"重新开始
   - **全天或部分时间：**（仅在选择单天时可用）
     - 关闭"全天"以申请部分时间
     - 为部分时间申请选择开始和结束时间
   - **原因：** 简要说明（可选但有帮助）
4. 点击**提交**
5. 将出现确认对话框，提醒您系统不跟踪剩余假期余额
6. 点击**我理解，提交申请**确认

### 检查申请状态

您的申请将显示以下状态之一：
- **待处理**（黄色）- 等待经理批准
- **已批准**（绿色）- 您的休假已确认
- **已拒绝**（红色）- 申请未被批准（查看管理员备注了解原因）

### 取消申请

您只能取消**待处理**的申请：
1. 找到待处理的申请
2. 点击**取消申请**
3. 确认取消

### 查看管理员备注

如果您的经理在申请中添加了备注：
1. 点击申请展开它
2. 查找"管理员备注"部分
3. 这可能解释批准条件或拒绝原因

## 物资申请

工作需要物资？在这里申请。

### 提交物资申请

1. 在导航中点击**物资**
2. 点击**申请物资**按钮
3. 填写表单：
   - **您需要什么？** - 输入您需要的物品（例如，"清洁用品"）
   - **产品链接：**（可选）粘贴确切产品的URL
   - **描述：**（可选）关于物品的额外详情
4. 点击**提交申请**

### 申请状态

- **待处理** - 等待批准
- **已批准** - 您的申请已批准（物资将被订购）
- **已拒绝** - 申请被拒绝（查看管理员备注）

### 取消物资申请

1. 找到您的待处理申请
2. 点击**取消**
3. 确认取消

## 儿童记录（仅限保姆/教师）

**此功能仅在您被分配为保姆或教师角色时可用。**

记录您照顾的儿童的日常活动。

### 访问记录

1. 如果您有访问权限，您将在导航中看到**记录**
2. 点击**记录**打开儿童记录页面

### 记录类别

您可以跟踪四种类型的活动：

| 类别 | 图标 | 描述 |
|------|------|------|
| **睡眠** | 月亮 | 午睡和就寝时间（带开始/结束时间） |
| **进食** | 刀叉 | 正餐和零食 |
| **换尿布** | 婴儿 | 换尿布 |
| **洗澡** | 淋浴头 | 洗澡和淋浴 |

### 创建新记录

1. 点击**新记录**标签（或它可能默认选中）
2. **选择儿童** - 点击儿童的名字（Zoe、Zara 或 Zander）
3. **选择类别** - 点击活动类型（睡眠、进食、换尿布或洗澡）
4. **日期** - 默认为今天，如需要可更改
5. **时间** - 输入活动的时间
   - 对于**睡眠记录**：您可以输入开始时间、结束时间或两者：
     - **两个时间都有**：显示为 "9:00 PM - 7:00 AM"（完整睡眠记录）
     - **仅开始时间**：显示为 "入睡 9:00 PM"（当孩子刚睡着时使用）
     - **仅结束时间**：显示为 "醒来 7:00 AM"（当您只知道醒来时间时使用）
   - 对于**其他记录**：输入发生的时间
6. **描述**（可选）- 添加任何备注
   - 对于进食记录，您会看到菜单建议，可以点击快速添加
7. 点击**添加记录**

### 查看记录

使用顶部的标签以不同方式查看记录：

- **新记录** - 创建新条目
- **全部** - 按日期排序查看所有记录
- **按儿童** - 按特定儿童筛选记录
- **按类型** - 按类别筛选记录（睡眠、进食等）

### 按日期筛选

查看记录时，您可以按日期范围筛选：
1. 使用顶部的日期选择器
2. 或点击快速筛选：**今天**、**本周**或**本月**

### 编辑记录

1. 找到您想编辑的记录
2. 点击右侧的**铅笔图标**
3. 进行更改
4. 点击**更新记录**

### 删除记录

1. 找到您想删除的记录
2. 点击右侧的**垃圾桶图标**
3. 确认删除

## 我的资料

查看和更新您的个人信息。

### 查看您的资料

1. 在导航中点击**资料**
2. 您将看到您的：
   - 姓名和角色
   - 头像
   - 电话号码
   - 出生日期
   - 紧急联系人
   - 重要日期（生日、纪念日等）

### 更新您的信息

1. 点击**编辑资料**按钮
2. 您可以更新以下内容：
   - **头像** - 点击您的照片上传新照片
   - **语言** - 选择英语、西班牙语或中文
   - **全名** - 您的显示名称
   - **电话号码** - 您的联系电话
   - **出生日期** - 您的生日
   - **紧急联系人** - 紧急情况下联系谁
   - **重要日期** - 添加您希望经理知道的生日、纪念日或其他日期
3. 点击**保存更改**

### 添加重要日期

重要日期帮助您的经理记住特殊场合：
1. 转到**资料** > **编辑资料**
2. 滚动到**重要日期**部分
3. 点击**添加日期**
4. 输入标签（例如，"孩子的生日"）并选择日期
5. 点击**保存更改**

## 提示与故障排除

### 一般提示

1. **每天检查任务** - 掌握新分配的任务
2. **及时完成任务** - 完成后立即标记为完成
3. **提前申请休假** - 给经理时间安排替代人员
4. **保持资料更新** - 特别是紧急联系人信息

### 应用不工作？

**尝试以下步骤：**

1. **强制关闭并重新打开应用**
   - iPhone：从底部向上滑动，将应用滑走
   - 重新打开应用

2. **检查您的网络连接**
   - 确保您有 WiFi 或蜂窝数据
   - 尝试加载网站测试

3. **更新应用**
   - 打开 TestFlight
   - 检查是否有可用更新
   - 如果有，点击**更新**

4. **清除登录并重试**
   - 退出登录然后重新登录
   - 检查您的邮箱获取登录链接

### 无法登录？

1. **检查您的电子邮件地址** - 确保与经理注册的一致
2. **检查垃圾邮件文件夹** - 登录邮件有时会到那里
3. **等待几分钟** - 有时邮件会延迟
4. **联系您的经理** - 他们可以验证您的邮箱是否正确

### 任务不显示？

1. **下拉刷新** - 在任务列表上向下滑动
2. **检查所有标签** - 任务可能在不同的状态
3. **检查您的网络** - 任务需要网络才能加载

### TestFlight 问题

**应用过期？**
- TestFlight 应用在90天后过期
- 等待您的经理发送新版本
- 有新版本时您会收到通知

**找不到应用？**
- 打开 TestFlight 并寻找 Household Manager
- 如果不在那里，请求新的邀请链接

### 需要帮助？

如果遇到以下问题，请联系您的经理：
- 登录问题
- 缺少任务或错误的分配
- 应用崩溃或错误
- 任何其他问题
